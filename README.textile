h1. Radig GearMailer

Plugin que permite o envio de emails utilizando Gearman e Amazon SES.

h2. Requisitos

* CakePHP 2.x
* "AWS SDK for PHP 2":https://github.com/aws/aws-sdk-php
* "Gearman":http://gearman.org/
* Gearman PHP Extension

h2. Instalação

* Clone, faça o download ou coloque como submódulo este repositório em Plugins
* Instale o aws-sdk em app/Vendor/AWS
* Instale o Gearman e sua extenção em PHP

h2. Uso

Crie uma configuração de email com a seguinte configuração:

<pre>
    public $default = array(
        'transport' => 'GearMailer.Gear'
    );
</pre>

Crie uma segunda configuração de email chamada 'gearman', configurando o transporte desejado (qualquer um disponível no seu sistema).
No exemplo abaixo, é usado o transporte Ses, disponível nesse plugin:

<pre>
    public $gearman = array(
        'transport' => 'GearMailer.Ses',
        'region' => 'us-east-1',
        'username' => 'sua chave AWS',
        'password' => 'sua chave secreta AWS',
        'dkim' => 'um email verificado' // Caso queira ativar o DKIM
    );
</pre>

Abaixo um exemplo utilizando um dos transportes padrões do CakePHP, Smtp:

<pre>
    public $gearman = array(
        'transport' => 'Smtp',
        'host' => 'email-smtp.us-east-1.amazonaws.com',
        'port' => 587,
        'tls' => true,
        'username' => 'sua chave SES',
        'password' => 'sua chave secreta SES',
    );
</pre>

Para ativar o Gearman Worker, que é quem efetivamente enviará o email, execute no terminal

bc. Console/cake GearMailer.EmailWorker

E deixe o console em execução.

Agora basta fazer um envio de email normal utilizando o *CakeEmail*

<pre>
    $email = new CakeEmail()
        ->from($from)
        ->to($to)
        ->subject($subject)
        ->send();
</pre>
